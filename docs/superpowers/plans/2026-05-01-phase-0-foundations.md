# Phase 0 — Foundations Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Stand up an empty-but-runnable React + Supabase shell with all foundational tooling (TS, Tailwind, shadcn, Router, TanStack Query, Zustand, i18n EN/EL, Vitest, Playwright, CI) so subsequent phases can plug into a working baseline.

**Architecture:** Vite SPA with TypeScript strict mode. Supabase backend (Auth/Postgres/Realtime/Storage) reached over HTTPS+WebSocket. State split into TanStack Query (server) + Zustand (client). UI built from shadcn/ui (Radix-based, copy-paste, Tailwind-styled). Bilingual EN/EL via react-i18next from day 1.

**Tech Stack:** Vite 5, React 18, TypeScript 5 (strict), React Router 6, Tailwind 3, shadcn/ui, TanStack Query 5, Zustand 4, react-i18next, @supabase/supabase-js 2, Vitest 1, React Testing Library 14, Playwright 1, GitHub Actions.

**Reference spec:** `docs/superpowers/specs/2026-05-01-itdevcrm-design.md`
